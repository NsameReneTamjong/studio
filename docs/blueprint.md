# **App Name**: EduNexus

## Core Features:

- User Authentication & Profiles: Secure user registration and login for Administrators, Teachers, and Students. Each user type will have a unique profile dashboard.
- Student Management: Admins can add, edit, and view student records, including personal information, enrollment details, and academic history.
- Teacher & Staff Directory: Admins can manage teacher and staff profiles, assign courses, and define their roles and permissions.
- Course & Class Management: Administrators can create courses, define class schedules, assign teachers, and enroll students into specific classes.
- Grade Book: Teachers can input and manage student grades for assignments, quizzes, and exams for their assigned classes.
- Attendance Tracking: Teachers can easily mark and track student attendance for each class session, providing a clear record of presence.
- AI-Powered Feedback Generation: Teachers can use an AI tool to generate personalized and constructive feedback for students based on their grade and attendance data.

## Style Guidelines:

- A light color scheme to convey clarity and professionalism, appropriate for an educational institution.
- Primary color: Deep institutional blue (#264D73) to represent stability and knowledge.
- Background color: Very pale blue-grey (#F0F2F5) for a clean and spacious interface.
- Accent color: Vibrant sky blue (#67D0E4) to highlight interactive elements and call-to-actions, ensuring visual distinction and focus.
- Body and headline font: 'Inter' (sans-serif) for its modern, clean, and highly readable design, suitable for data-heavy applications and professional content.
- Use clear, concise, and universally recognizable line icons that enhance navigation and functional clarity without visual clutter.
- Adopt a clean, structured, and responsive layout featuring prominent dashboards for each user role, ensuring easy access to relevant information and tasks.
- Implement subtle, fast, and smooth transitions for state changes and form submissions to provide a fluid user experience without distraction.
